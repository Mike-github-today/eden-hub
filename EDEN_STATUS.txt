ARCHITECTURE NOTES
--------------------------------------------------------------------------------

### Eden Leaf SVG Design
The Eden Leaf SVG uses:
- Outlined style (no fill, stroke only)
- Main outline path: curved leaf shape
- Central vein: vertical line
- Side veins: curved lines at 0.3 opacity
- All strokes: #2d5f4d (Eden Deep)
- Stroke width: 1.5px main, 1px details

### Breathing Animation
The eden-breathe animation:
- Duration: 4 seconds
- Easing: ease-in-out
- Scale: 1.0 -> 1.08 -> 1.0
- Opacity: 0.7 -> 1.0 -> 0.7
- Infinite loop

### Role Detection
EmptyState uses the global currentUserRole variable:
- getRole() returns 'admin' or 'associate'
- Messages automatically update based on role
- Action buttons respect role permissions
